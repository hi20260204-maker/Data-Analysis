# Data Analysis with Basic Statistics
## 데이터분석 또는 머신러닝을 위한 기초통계

**표준편차를 통해 지역별 수확량의 변동성을 예측할 수 있다**
>>수확량의 변동이 심한 곳이 단순합을 하면 값에 대한 외곡이 생길 수 있음 : 오차의 크기를 잘 못 구할수도 
- 해결책 음수를 제거하고 평균을 기준으로 값을 보자 = **표준편차를 이용**<br>

```
 s_x = \sqrt{\frac{1}{n} \sum (x_i - \bar{x})^2} 
```

정규분포에서는 평균을 중심으로 대칭적으로 값들이 분포하며, 평균에서 멀어질수록  빈도가 줄어듭니다. 
> 표준편차의 한계 : 극단적인 값에 대해서는 알 수 없음<br>
>>해결책 1. 그림으로서 시각화해서 극단적인 값까지도 표현

수확량에 영향을 주는 변수 정의
> 독립변수 : 종속변수에 영향을 주는 변수<br>
- 독립변수 중 조절을 못 하는 부분도 있음 -> 이럴때는 조절할 수 있는 부분을 조절함<br>
> 종속변수 : 독립변수를 조절할 때 그에 따라서 변화되는 변수

상관관계 : 비료의 양과 수확량 간의 상관관계를 분석해보니, 비료를 많이 줄수록 수확량이 증가하는 경향이 있음<br>

- 독립변수 중 영향을 줄 수 있는 요인을 조절하여 결과 값을 가진다.
- 양의 상관관계와 음의 상관관계가 있다.<br>

## p-값
- 불확실성을 줄이기 위해서 p-값이 필요
- 정의 : 우리의 가설이 틀렸다고 가정할 때, 관찰된 결과(또는 그보다 더 극단적인 결과)가 우연히 발생할 확률을 나타냄
- But p-값이 너무 작으면 (p-값 <0.05) -> 귀무가설을 기각
